# שאלון ידע בשוק ההון

שאלון אינטראקטיבי לבדיקת רמת הידע בשוק ההון, הכולל 100 שאלות בדרגות קושי שונות.

## תכונות

- 100 שאלות במאגר
- 10 שאלות אקראיות בכל סבב
- טיימר לכל שאלה
- מערכת שלבים
- משוב מיידי עם אמוג'ים
- תמיכה מלאה בעברית
- עיצוב רספונסיבי

## הוראות הפעלה

1. העתק את הריפוזיטורי:
```bash
git clone https://github.com/your-username/investment-quiz.git
```

2. פתח את הקובץ `index.html` בדפדפן או העלה לשרת.

## טכנולוגיות

- HTML5
- CSS3
- JavaScript (Vanilla)
